Flash the Embroidery Stick firmware

Works in Chrome or Edge on a computer with a USB port (WebSerial isn't supported in Firefox/Safari). No software install needed.

  1. Plug the stick into a USB port.
  2. Click the button below.
  3. Pick the right serial port when prompted, then follow the on-screen steps.

First time? After flashing, hold the button on the stick for 3 seconds at power-on to set up WiFi — see the README.
